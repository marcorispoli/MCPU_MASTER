\doxysubsection{Detailed Description}
This is the status register of the Arm motorization. 

There are several features that this register allow to handle\+:
\begin{DoxyItemize}
\item Arm Angle verification option\+: enables or disables the test on the angle validity during exposure;
\item Arm target angle and projection setting\+: allow to handles the arm positioning 
\end{DoxyItemize}
